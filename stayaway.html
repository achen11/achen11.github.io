Stay Away Project

Skills: HTML/CSS, JavaScript

Role: Frontend Developer

Description:

We wanted to make a fun adventure game that was Covid-19 themed. First we designed the look of the game and drew out the settings and characters using pixel art.

Then we figured out game design aspects. We coded the characters in JavaScript so they could move using the arrow buttons.

We designed moving obstacles that the character had to dodge and jump around.

We also designed and connected our game page with an intro page and ending page
