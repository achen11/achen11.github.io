Minesweeper Project

Skills: C++

Role: Frontend Developer

Description:

As big fans of the game Minesweeper we wanted to try recreating the game from scratch. We used C++ and the ge:211 graphical library to code the project. You can see a playthrough of our game below.

One of our hardest challenges was figuring how to structure our code. We ended up splitting our code into 4 seprate areas: a Board, Model, Controller and View sections.

In addition to the backend code which randomly generates the locations of mines and the numbered tiles surrounding the mines, we also incorporated taking in user input.

If you right-click on the tiles you can unveil them and if you left-click on the tiles you can flag them. When you successfully unveil all the non-mine tiles you win the game!
